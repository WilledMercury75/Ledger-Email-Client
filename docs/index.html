API Endpoints

Build integrations with the JSON REST API on port 8420.

GET /api/identity Your Ledger ID & public key
GET /api/messages List messages by folder
POST /api/messages Send encrypted message
GET /api/peers Connected P2P peers
POST /api/peers Connect to peer via multiaddr
GET /api/settings Delivery mode & config
PUT /api/settings Update settings
POST /api/gmail/config Configure Gmail bridge
POST /api/gmail/fetch Fetch Gmail inbox
DELETE /api/messages/{id} Delete a message
